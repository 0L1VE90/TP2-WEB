1. Structure de base

Le code JavaScript peut être intégré directement dans la page HTML via la balise <script>.

2. Les Commentaires

Ils permettent une relecture du programme (travail en équipe).

  • // : Commentaire sur une seule ligne.
  • /* ... */ : Commentaire multi-lignes.

3. Déclarations et Variables

JavaScript est un langage faiblement typé. On utilise principalement let pour déclarer des variables locales.

Les 8 types de base incluent : Number, String, Boolean, Null, Undefined, BigInt, Object, et Symbole.

4. Les Opérateurs

Il existe trois catégories : Arithmétiques (+, -, *, /, %, **), Logiques et d'Affectation.
